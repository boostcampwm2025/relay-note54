## AI를 활용할 수 있는 방법

### 1.하루 미션 수행 시간을 정리(최소 수면 시간 보장)
  - 개인 일정
  - 수면 시간
  - CS 학습
  - 구현 시간

### 2.학습 정리 기반으로 체크 문항 생성
  - 정리한 자료 베이스로 문제 생성
  - 문제를 풀어보며 부족한 부분 학습

### 3.메뉴 추천
  - 점심
  - 저녁
  - 야식

### 4.주말 이벤트 생성
  - 휴식 방법
  - 나만의 체크리스트 기반 필요한 부분 정리
  - 영화 추천
  - 드라마 추천

### 5.슬랙 묻고답하기 등 에서 중복 질문 찾아보기
  - 유사한 질문 찾기

### 6.AI로 내 건강 상태 확인하기
  - 개인 컨디션 관련 설문 문항 생성
  - 문항에 따른 현재 컨디션 작성
  - 설문 분석

### 7.어제의 사회 이슈 검색
  - 흥미로운 이슈를 기반으로 피드백 시간에 스몰토크로 시작하기

### 8.슬랙 채널별 활성화
  - 다른 채널에서 함께 토론해볼 문서 찾기
